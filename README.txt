ClicketyClack

A Railway-builder incremental game for the web browser.

USAGE:
Open index.html in a web browser.
	A publicly available, and usable page is available via github:
		https://ticklishhoneybee.github.io/ClicketyClack/index.html
Start clicking and build your railway empire!

Additionally, you can build your own incremental game by replacing
 'clicketyclack.js' with your own game data.
